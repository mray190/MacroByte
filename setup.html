Setup

The micro:bit is a tiny computer that anyone can program. The board comes with lights, buttons, and a wireless radio, which we’ll use to send messages from one micro:bit to another. If you haven’t used the micro:bit before, check out the Getting Started [link] guide from the micro:bit website.

You’ll need two micro:bit boards, one to send a message and one to receive it. We’ll be using the JavaScript development tools [link] in this tutorial. You can also download code [link] for micropython.
